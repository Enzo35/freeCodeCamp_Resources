Access Modifires In JAVA

An access modifier is used for attributes, methods and constructors to only make them accessible to a certain extent.

In this page I will be using 'attributes' as catch all term for classes, variables, methods and constructers

Private

This modifier makes that the attribute can only be accessed in the class it was created.

Uses the "private" keyword.

  • private class Main {}

  • private String var = "This string is a private attribute"

    • Default

    This is an access modifier a level bellow the "protected" modifier.

    In this modifier a attribute can be accessed only by members of the same package.

    The default modifier does not have a keyword.

  • class Main {}

  • int year = 2010

    • Protected

    This is an access modifier a level bellow the "protected" modifier.

    This is the same as the "default" modifier with the addition of child classes outside of the package are also given access to the attribute.

    Uses the "protected" keyword.

  • protected string = "Protected String"

    • Public

    A public attribute can be accessed by any other at any time.

    Uses the "public" keyword.

  • public class Main {}

    • Honorable Mention

  • Static

    • Not a access modifier

    The "static" keyword makes a atribute to be of the class, not the object. This means only one copy of it exists for the entire class, rather than each having its own copy.

    Only static methods can access static atributes
